# 🌱 Consumo Consciente e Sustentabilidade

Projeto Integrador desenvolvido para promover a conscientização sobre consumo responsável, reciclagem, economia de recursos naturais, descarte correto de resíduos eletrônicos e obsolescência programada.

## 🚀 Acesse o projeto

🔗 [https://leticia-sanders-ux.github.io/projeto_integrador/](https://leticia-sanders.github.io/projeto_integrador/)

## 📋 Funcionalidades

- Informações sobre consumo consciente
- Dados ambientais globais
- Seção sobre obsolescência eletrônica
- Diagnóstico de hábitos sustentáveis
- Calculadora de consumo de água
- Conteúdo complementar em vídeo
- Desafio de conhecimento ambiental com pontuação
- Sistema de árvores virtuais por acerto
- Layout responsivo para celular, tablet e desktop

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Incentivar práticas sustentáveis e demonstrar como pequenas mudanças de comportamento podem gerar impactos positivos para o meio ambiente e para a sociedade.
